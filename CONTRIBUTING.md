# Contributing to CodeThinker

Thank you for your interest in contributing! 🎉

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/codethinker.git`
3. Install dependencies: `npm install`
4. Copy env file: `cp .env.example .env.local` and add your `GROQ_API_KEY`
5. Start dev server: `npm run dev`

## Development Guidelines

### Code Style
- TypeScript strict mode — no `any` types
- Tailwind CSS for styling
- Functional components with hooks
- Keep components small and focused

### Commit Messages
Follow [Conventional Commits](https://conventionalcommits.org):
- `feat:` — new feature
- `fix:` — bug fix
- `docs:` — documentation
- `refactor:` — code refactor
- `chore:` — maintenance

### Pull Request Process
1. Update README if you add new features
2. Ensure `npm run build` passes
3. Add a clear description of what the PR does

## Areas to Contribute

- 🔧 New thinking modes
- 🤖 Additional AI provider integrations
- 🌍 i18n / translations
- 🎨 UI improvements
- 📚 Documentation
- 🐛 Bug fixes
